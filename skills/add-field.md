# Skill: add-field

## Goal
Extend content schema with a new field in a maintainable way.

## Workflow
1. Create migration for schema change (versioned, deterministic).
2. Extend domain/application DTO validation and mapping.
3. Update admin form template and submit handling.
4. Add repository and application tests.
5. Update docs (`content-model.md`, `admin-rules.md`) to describe the new field behavior.
